6. Instrumentos de evaluación

Registro de caritas

El instrumento sirve para evaluar el criterio 3.7 del área III: Expresarse de manera creativa, utilizando diversas herramientas o aplicaciones digitales intuitivas y visuales para expresarse de manera creativa.

Está diseñado para que cada alumno/a se autoevalúe tras el uso de todas las herramientas y recursos TIC de la SdA. Cada uno tendrá que colorear de rojo o verde la carita triste o contenta según considere si se ha expresado de manera creativa o no con las distintas herramientas como el robot, el uso de la PDI o la grabación de audios.

Tomaré notas durante las diversas sesiones para orientar a los alumnos si les surgieran dudas a la hora de autoevaluarse con algunas de las herramientas. También existe la posibilidad de usar el instrumento inmediatamente después de usar cada recurso para que así tengan fresco el trabajo realizado y puedan autoevaluarse de una manera más adecuada y certera. 

Estos resultados me servirán para conocer la motivación y disfrute del alumnado con las diversas herramientas TIC, para así ir concretando las que usaremos en futuras SdA. 

Semáforo de evaluación

El instrumento sirve para evaluar el criterio 2.6 del área II: Participar en proyectos utilizando dinámicas cooperativas, compartiendo y valorando opiniones propias y ajenas, expresando conclusiones personales a partir de ellas...

Está diseñado para que los alumnos/as se coevalúen. En pequeños grupos, un compañero tendrá que evaluar la participación de otro en la actividad relacionada con este criterio, pintando de color rojo, amarilo o verde el semáforo según proceda. Debajo del semáforo se escribirá el nombre del alumno evaluado en ese semáforo.

Estos resultados me pueden servir para conocer la percepción del alumnado sobre el trabajo y la participación de sus compañeros. Yo realizaré también una heteroevaluación del mismo criterio y así podré comparar ambos resultados.
